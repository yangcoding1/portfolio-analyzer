# 📄 Product Requirements Document (PRD)

**프로젝트명:** Neo-Visualizer (개인화 포트폴리오 백테스트 웹 애플리케이션)

## 1. 프로젝트 배경 및 목적
* **배경:** 기존 포트폴리오 백테스트 툴은 UI/UX가 직관적이지 않고, 개인화된 분석 지표나 커스텀 로직을 추가하기 어려움.
* **목적:** 트렌디하고 직관적인 UI를 제공하며, 사용자가 원하는 맞춤형 자산군과 백테스트 로직을 자유롭게 확장하고 테스트할 수 있는 개인화된 분석 환경 구축.

## 2. 타겟 사용자 및 개발자 컨텍스트 (For AI Agent)
* **타겟 사용자:** 데이터 기반 투자를 지향하는 개인 투자자.
* **개발자 프로필:** 코딩 초보자이며, AI 에이전트(Claude, Gemini 등)를 적극 활용하는 **바이브 코딩(Vibe Coding)** 방식으로 프로젝트를 완성할 예정.
* **AI 에이전트 행동 지침:**
    1. 코드를 제시할 때는 초보자도 쉽게 이해하고 복사/붙여넣기 할 수 있도록 파일 경로와 함께 전체 코드 블록을 제공할 것.
    2. 복잡한 개념은 직관적인 비유를 들어 단계별로 설명할 것.
    3. 확장을 고려해 모듈화된 코드를 작성하고, 하드코딩을 지양할 것.

## 3. 주요 기능 요구사항 (MVP - Minimum Viable Product)

### 3.1 자산 및 포트폴리오 설정
* 티커 검색 및 자동완성 입력.
* 자산별 투자 비중(%) 입력 (총합 100% 검증 로직).
* 초기 투자금(Initial Amount), 리밸런싱 주기(월/분기/연), 배당 재투자 여부 등 테스트 조건 설정.

### 3.2 백테스트 엔진 (Core Logic)
* 기본 성과 지표 산출: CAGR(연평균 수익률), MDD(최대 낙폭), Sharpe Ratio, Volatility(변동성), Best/Worst Year.
* 벤치마크(SPY 등) 성과 비교.

### 3.3 대시보드 및 데이터 시각화 (UI/UX)
* **포트폴리오 성장 차트:** 시간 경과에 따른 자산 가치 변화 라인 차트 (Zoom/Pan 지원).
* **낙폭(Drawdown) 차트:** 시기별 고점 대비 하락률 시각화.
* **자산 비중 파이 차트:** 설정한 포트폴리오 비중 시각화.

## 4. 제안하는 기술 스택 (Tech Stack)
* **Frontend:** Next.js (트렌디한 대시보드 UI 및 차트 라이브러리 활용)
* **Backend:** FastAPI (Python 기반의 빠른 비동기 API 및 데이터 연산)
* **Database:** TimescaleDB (금융 시계열 데이터 저장 및 조회 최적화)
* **Infrastructure:** Docker & WSL2 (로컬 환경 컨테이너화 및 쉬운 배포)

## 5. 소프트웨어 아키텍처 (Clean Architecture)
이 프로젝트는 향후 무한한 기능 확장을 위해 **클린 아키텍처(Clean Architecture)** 계층 구조를 엄격하게 따릅니다. AI 에이전트는 아래의 의존성 규칙(`Presentation -> Application -> Domain <- Infrastructure`)을 준수하여 코드를 설계해야 합니다.

* **Domain Layer (도메인):** 외부 의존성이 전혀 없는 핵심 비즈니스 로직과 엔티티 (예: 자산, 포트폴리오 모델, CAGR/MDD 계산 순수 함수).
* **Application Layer (애플리케이션):** 도메인 객체를 활용해 사용자의 요청을 처리하는 유스케이스 (예: '포트폴리오 백테스트 실행' 파이프라인).
* **Presentation Layer (프레젠테이션):** 사용자 인터페이스 및 외부 요청 처리 (예: Next.js UI 컴포넌트, FastAPI 라우터/컨트롤러). 오직 Application 계층에만 의존함.
* **Infrastructure Layer (인프라스트럭처):** 외부 시스템과의 통신 및 데이터 영속성 (예: TimescaleDB 연동, yfinance 등 외부 주가 API 호출). 오직 Application/Domain 계층의 인터페이스를 구현하는 형태로 의존함.

## 6. 향후 확장 기능 (Phase 2)
* 매크로 경제 지표(금리, 환율) 연동 분석 추가.
* 백테스트 결과 기반 LLM 투자 인사이트 요약 리포트.
* 다중 포트폴리오 저장 및 동시 비교 기능.
